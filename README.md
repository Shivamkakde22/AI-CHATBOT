# AI Chatbot (Flask + PostgreSQL + Anthropic)

## Description
This is a full-stack AI chatbot built using Flask, PostgreSQL, and Anthropic API.  
It stores chat history and generates real-time AI responses.

---

## Features
- Real-time AI chatbot
- Chat history stored in PostgreSQL
- Flask backend with REST architecture
- Secure API key handling using .env
- Error handling for API & database

---

## Tech Stack
- Python
- Flask
- PostgreSQL
- Anthropic API
- HTML, CSS
